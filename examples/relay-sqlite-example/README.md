# gun-relay-sqlite

Manager+Adapter Stable version

Support iOS & Android 

# Quickly build a relay manager and Gun storage adapter with sqlite persistence.

```base
yarn install
```

```base
yarn build
```

```base
npx cap sync
```

```base
npx cap open ios | android  
```
